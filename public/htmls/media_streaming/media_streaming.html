Media Streaming API

You can use these REST APIs for Akamai® HD Network: Utility, Domain, Stream, and Archive Management. The following sections explain how to use these APIs to build a customized interface: Utility APIs, Domain APIs, Stream APIs, Archive Management APIs

and provide several common examples of their use - Example 1: Creating a Domain Workflow, Example 2: Creating a New Domain Version Workflow, Example 3: Activating a Domain Version on Staging or Production, Example 4: Creating a New Stream Workflow, Example 5: Editing an Existing Stream Workflow, Example 6: Archive Management/Editing an Event

Note the following:

- Currently, the APIs support only Akamai HD for the Adobe® HTTP Dynamic Streaming and Apple® HTTP Live Streaming.

- All the APIs might not work for every user. The APIs work based on the permissions available to the logged in user.

- For all APIs, when passing content as XML, please set the Content-Type HTTP header to application/xml.

- You must look at the detailed API documentation and the XMLs associated with the output and input payload. Look for the Type documentation associated with each XML element, and you will find more valuable documentation about individual elements.

- All the APIs work similar to the functionality available through the existing Luna Control Center.
