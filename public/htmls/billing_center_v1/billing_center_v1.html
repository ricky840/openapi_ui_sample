The Billing Center V1 API

This guide details the original version 1 of the Billing Center API. This API version has been deprecated. For the most recent version 2, see the Billing Center API.

The API allows you to access your monthly Akamai usage data in CSV format, aggregating it per contract or per reporting group. See the Reference section for details on available resources.
