Diagnostic Tools API

The Diagnostic Tools API offers programmatic access to a subset of Akamai’s diagnostic toolset available in Luna Control Center. With these tools you can self-diagnose and solve many common problems Akamai customers experience.

The Diagnostic Tools API responds to requests with standard HTTP status codes. When there’s a validation failure or when a resource fails to process your request, the API returns error messages in the HTTP Problem JSON format.
