Select the Group(Akamai Korea-C-1BBSI2U) which have authorization for using API and click the Menu(CONFIGURE - Manage APIs)
Click 'Add new collection...' below CCU APIs
Add new collection name and click 'Save' button
Click 'Add New Client' below new collection
Agree the 'akamai api livense agreement'
Add new client name and click 'Save' button
Click 'Authorizations' and 'New Authorizations'
Fill out the new authorization name and click 'Next' button
Check the grant settings and click 'Next' button
Set up the grant's settings and click 'Save' button
If the Status change 'In progress' to 'Active', it means it is ready to using API
You can download your token(.txt file) which include credential information. Now you are ready to use OPEN API Test Page with your own Token :)
